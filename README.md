# Генерация синтетического графового датасета для кредитного скоринга с помощью LLM

Репозиторий содержит материалы курсовой работы на тему: **«Разработка метода генерации графового набора данных для задачи кредитного скоринга с использованием LLM»**.

## О проекте

Классические модели кредитного скоринга оценивают компании изолированно, игнорируя системные риски, цепочки поставок и макроэкономические факторы. Графовые нейросети (GNN) способны решать эту проблему, однако исследования в этой области тормозятся из-за строгой конфиденциальности реальных банковских данных.

**Цель проекта:** Разработка пайплайна для создания синтетических (но экономически достоверных) графовых датасетов на базе открытых данных (ФНС, Росстат) с использованием локальных Больших Языковых Моделей (LLM) в качестве AI-андеррайтеров.

## Методология и Пайплайн

Проект состоит из 4 основных этапов:

1. **Дата-вербализация и Эмбеддинги:**
   * Сбор панельных финансовых данных 10 000 российских компаний из 5 макрорегионов (2017–2024 гг.).
   * Перевод сырых финансовых метрик (выручка, капитал, долг) в текстовые профили-описания.
   * Векторизация профилей с помощью `Qwen3-Embedding-0.6B`.

2. **Построение экономического графа:**
   * Формирование графа подобия на основе косинусного расстояния с использованием алгоритма k-NN (k=6).
   * Итоговый граф: 10 000 узлов, ~42 500 ребер, сохраняющий естественную структурную гомофилию (выявлены моноотраслевые и монорегиональные кластеры).

3. **Синтез целевой переменной с помощью LLM:**
   * Использование локальной модели `Qwen3-8B` (через API) в роли senior-андеррайтера корпоративного бизнеса.
   * Контролируемая генерация: на базе финансовых трендов LLM предсказывает риск дефолта (0 или 1) и генерирует текстовое обоснование (бизнес-гипотезу).
   * Борьба с "галлюцинациями" (например, влияние электромобилей на СТО в регионах) с помощью жестко заданного контекста и риск-правил.
   * Результат: сбалансированный датасет (52.3% дефолтов / 47.7% выживаний).

4. **Валидация (GraphSAGE):**
   * Обучение индуктивной графовой нейросети (`GraphSAGE` из PyTorch Geometric) на сгенерированном датасете.
   * Метрики на тестовой выборке: **Test Accuracy = 0.86**, **Test ROC-AUC = 0.94**. Высокие метрики доказывают, что синтезированная разметка подчиняется строгой экономической логике и пригодна для ML-задач.
